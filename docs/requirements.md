# 뽐뿌 광고 관리자 (Ppomppu Ad Admin) 요구사항 정의서

본 문서는 뽐뿌 익스텐션 개발 과정에서 발생한 주요 요구사항과 구현 내용을 정리합니다.

## 1. 프로젝트 개요
뽐뿌(`ppomppu.co.kr`) 사이트 내의 광고 위치를 시각화하고, 각 광고에 사용된 태그 정보를 확인하기 위한 크롬 익스텐션 관리 도구입니다.

## 2. 주요 기능 및 요구사항 (기능 이력)

### v1.0: 프로젝트 초기화 및 기본 감지
*   **광고 레이아웃 시각화**: 뽐뿌 내 광고 영역에 테두리와 라벨을 표시하여 위치를 가시화함.
*   **Google Ads 감지**: GPT(Google Publisher Tag) 및 AdSense 태그를 자동으로 찾아내어 표시함.
*   **기타 광고 감지**: 뽐뿌 자체 광고 슬롯 및 외부 네트워크 광고를 감지함.
*   **팝업 UI**: 익스텐션 아이콘 클릭 시 광고 가시화 On/Off 토글 기능 제공.

### v1.1: 사이드바 패널 전환
*   **사이드바 UI 리팩토링**: 팝업 방식에서 크롬 전용 사이드 패널(Side Panel) 방식으로 전환하여 페이지 탐색 중에도 지속적으로 정보를 확인할 수 있도록 개선.
*   **실시간 카운팅**: 현재 페이지에서 감지된 총 광고 수, Google Ads 수, 기타 광고 수를 실시간으로 집계하여 표시.

### v2.0: 상세 정보 및 상호작용 강화
*   **태그 리스트 표시**: 감지된 광고들의 고유 ID 또는 슬롯 명칭을 사이드바 하단 리스트로 제공.
*   **스크롤 및 강조 기능**: 리스트의 태그를 클릭하면 페이지 내 해당 광고 위치로 부드럽게 스크롤 이동하며, 노란색 펄스(Pulse) 애니메이션으로 강조 표시함.
*   **데이터 동기화 개선**: 동적 로딩(Lazy Loading)되는 광고를 MutationObserver를 통해 실시간으로 포착하여 리스트에 반영.

### v2.3: 광고 규격 표시 추가
*   **가로x세로 사이즈 노출**: 모든 광고 태그 명칭 뒤에 해당 영역의 실제 크기(예: `(300x250)`)를 자동으로 병기하여 표시함. 이 정보는 페이지 내 오버레이와 사이드바 리스트 모두에 적용됨.

## 3. 프로젝트 구조
*   `manifest.json`: 익스텐션 설정 및 권한 정의.
*   `content_script.js`: DOM 스캔, 광고 감지, 시각화 레이어 삽입 및 통신 로직.
*   `sidepanel.html/js`: 사이드바 UI 및 사용자 조작 처리.
*   `background.js`: 사이드바 실행 및 익스텐션 라이프사이클 관리.
*   `icon.png`: 익스텐션 대표 아이콘.
