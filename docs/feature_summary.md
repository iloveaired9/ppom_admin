# PPomppu Admin 기능 명세 요약 (v1.0.0)

지금까지 구현된 모든 핵심 기능과 요구사항을 탭별로 정리한 명세서입니다.

---

## 1. 광고 탭 (광고 정보 및 가시화)
- **실시간 광고 탐지**: 페이지 내 모든 광고 슬롯을 탐지하고 플랫폼별(Google, Kakao, Naver, Way2G 등) 수치를 집계합니다.
- **광고 가시화 (Inspector)**: ON 상태에서 페이지 내 광고 영역을 유색 배경과 테두리로 강조하여 위치를 직관적으로 보여줍니다.
- **대체 광고 탐지**: 광고 매칭 실패 시 노출될 대체 광고 설정(`FALLBACK_CONFIG`)을 분석하여 플랫폼 및 크기 정보를 목록에 표시합니다.
- **슬롯 이동**: 감지된 배너 리스트를 클릭하면 해당 광고 슬롯의 위치로 부드럽게 스크롤됩니다.

## 2. 링크스캔 탭 (외부 링크 탐지)
- **외부 도메인 전용**: 도메인이 없는 상대 경로(`/`)나 `#`, 스크립트 등을 제외하고, 순수하게 외부로 연결되는 모든 절대 경로(`http`, `https`, `//`, `\/`)를 추출합니다.
- **내부 필터링**: `ppomppu.co.kr`과 관련된 모든 내부 링크를 제외하여 순수 외부 도메인 연결 정보만 집중 제공합니다.
- **상세 정보 노출**: 각 링크 항목 하단에 **실제 URL(href)**을 표시하여 사전 확인이 가능합니다.
- **통합 관리**: 비정상 링크와 **모든 외부 링크**를 구분하여 집계하고 리스트를 제공합니다.

## 3. 뽐링크 탭 (뽐뿌 전용 링크 디코더)
- **s.ppomppu 탐지 및 디코딩**: `s.ppomppu.co.kr` 형태의 타겟 링크를 찾아 원본 URL로 디코딩합니다.
- **유틸리티 버튼**: 각 링크별로 **복사(Copy)** 및 **열기(Open)** 버튼을 제공하며, 복사 성공 시 체크(✓) 피드백을 제공합니다.
- **강력한 하이라이트**: 리스트 마우스 오버 시 본문의 해당 링크를 노란색 배경과 빨간 테두리로 강조합니다.
- **DEBUG CONSOLE**: 통신 상태와 탐지 로그를 실시간으로 보여주는 가변 패널을 제공하며, **접기/열기** 기능을 통해 UI 공간을 효율적으로 사용합니다.

## 4. 기타 탭 (설정 및 정보)
- **기본 시작 탭 설정**: 익스텐션 실행 시 처음으로 보여줄 초기 탭(광고/링크스캔/뽐링크)을 설정할 수 있습니다.
- **버전 정보**: 현 버전 1.0.0 정보 및 개발자 정보를 확인 가능합니다.

---

## 5. 공통 및 백그라운드 기능
- **이름 변경**: 기존 'Ppomppu Ad Admin'에서 **'PPomppu Admin'**으로 브랜딩을 통합했습니다.
- **탭 동기화**: 탭 전환 및 페이지 새로고침 시 모든 가시화 상태와 데이터가 누락 없이 동기화됩니다.
- **완벽한 클린업**: 사이드패널을 닫거나 다른 탭으로 이동 시, 페이지에 남은 모든 강조 효과(하이라이트)를 즉시 제거하여 웹 페이지를 원래 상태로 복구합니다.
- **상태 배지**: 헤더의 배지를 통해 현재 뽐뿌 사이트 접속 여부 및 익스텐션 작동 상태(ON/OFF/N/A)를 실시간으로 표시합니다.
