## 주요 개선 사항

### 1. 대체 광고 탐지 및 표시 (신규)
- **Fallback 탐지**: `test/slot_handler.js`의 설정을 바탕으로, `list_f`, `main_f` 등 주요 광고 슬롯의 대체 광고 정보(카카오 애드핏, WTG 등)를 자동으로 감지합니다.
- **UI 시각화**: 사이드패널 광고 목록에서 "대체: 300x250 (Kakao)"와 같이 배지 형태로 정보를 표시하여, 기본 광고 실패 시 노출될 정보를 미리 확인할 수 있습니다.

### 2. 강력한 탭 동기화
- **리액티브 싱크**: 탭 전환 및 URL 업데이트에 실시간으로 반응하도록 로직을 개선했습니다.
- **Tabs 권한 활용**: 익스텐션 권한을 강화하여 현재 활성화된 탭의 정보를 더 정확하게 파악합니다.

### 3. 버전 관리 및 태깅 (v1.0.0)
- **버전 통일**: `manifest.json` 및 UI의 버전 정보를 `1.0.0`으로 통일했습니다.
- **Git 태그**: 현재의 안정된 상태를 `v1.0.0` 태그로 고정했습니다.

### 10. 디버그 패널 토글 기능 추가 (신규)
- **UI 개선**: 뽐링크 탭의 디버그 콘솔 헤더에 **접기/열기** 버튼을 추가했습니다.
- **가독성 확보**: 기본적으로 로그 영역을 접어두거나 펼칠 수 있어, 뽐링크 리스트를 더 넓게 확인할 수 있습니다.

- [x] **상태 유지**: 로그가 접힌 상태에서도 데이터는 내부적으로 계속 누적되며, 다시 '열기' 시 모든 로그를 확인할 수 있음.
- [x] **사용자 편의성**: 불필요한 디버그 UI 노출을 최소화하여 기본 기능을 편리하게 사용할 수 있도록 함.
